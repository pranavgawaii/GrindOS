1. Three Types of Class Methods

Python classes contain three distinct method scopes, defined by decorators:

Method Type Decorator First Parameter Capabilities
Instance Method None. `self` Can access and modify instance state and class state.
Class Method `@classmethod` `cls` Can access and modify class-level state. Cannot access instance state.
Static Method `@staticmethod` None. Isolated function namespaces. Cannot access class or instance state.

2. Code Examples and Use Cases

class User:
    user_count = 0 # Class variable
    
    def __init__(self, username):
        self.username = username
        User.increment_count() # Call classmethod
        
    @classmethod
    def increment_count(cls):
        # Access and modify class state via cls
        cls.user_count += 1
        
    @staticmethod
    def validate_password(password):
        # Pure utility helper
        return len(password) >= 8

🎯 Topic Q&A

Q1: What is the main difference between `@classmethod` and `@staticmethod`?

A class method takes `cls` (the class object) as its first implicit parameter and can read/modify class-level variables. A static method takes no implicit first parameters, acting like a normal isolated utility function bounded to the class namespace.

Q2: When would you use a `@classmethod` instead of an instance method?

You use a class method to define alternative constructors (factory methods) to instantiate objects from different formats (like JSON, strings, or DB rows) or when managing global class-wide state.

Q3: Why use `@staticmethod` instead of writing a standalone global function?

Static methods group logical utility functions inside a class namespace, which makes the codebase highly organized, clean, and self-documenting.

Q4: Can a static method invoke a class method in Python?

Yes. Although it lacks `cls`, a static method can invoke a class method by explicitly referencing the Class name (e.g., `ClassName.classmethod_name()`).

Q5: Write a factory classmethod creating a Person from birth year. 
import datetime

class Person:
    def __init__(self, name, age):
        self.name, self.age = name, age
    @classmethod
    def from_birth_year(cls, name, birth_year):
        curr_year = datetime.datetime.now().year
        return cls(name, curr_year - birth_year)
