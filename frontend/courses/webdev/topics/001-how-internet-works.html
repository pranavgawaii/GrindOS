1. The Internet vs The Web

The Internet is the physical infrastructure — millions of computers linked by cables, fiber, and wireless signals. The World Wide Web is just one service running on top of the internet, using HTTP to share documents and data.

2. How Data Travels

3. IP Addresses

Every device on the internet has an IP address — a unique numeric identifier.

VersionFormatExample
IPv432-bit, dotted decimal192.168.1.1
IPv6128-bit, hexadecimal2001:0db8::1

4. DNS — The Internet's Phone Book

When you type google.com, DNS translates it to an IP address your computer can route to.

1. Browser checks local cache
2. Queries Recursive Resolver (your ISP)
3. Resolver queries Root Nameserver
4. Root refers to .com TLD server
5. TLD refers to google.com's Authoritative NS
6. Returns IP: 142.250.195.174
7. Browser connects to that IP

5. TCP/IP — The Reliable Transport

HTTP rides on top of TCP (Transmission Control Protocol), which guarantees:

  • Packets arrive in order
  • Lost packets are re-transmitted
  • Data integrity is verified (checksums)
